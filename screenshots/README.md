## Adding Screenshots

To add screenshots:
1. Take a screenshot of the application
2. Save it as a PNG file with a descriptive name
3. Place it in this folder
4. Update this README with the screenshot reference

Example:
```markdown
![Home Page](./home-page.png)
```
